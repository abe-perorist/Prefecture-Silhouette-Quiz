このフォルダには以下の都道府県シルエット画像を配置してください：

必要な画像ファイル：
========================

1. hokkaido_silhouette.png   - 北海道のシルエット
2. kyoto_silhouette.png      - 京都府のシルエット
3. tokyo_silhouette.png      - 東京都のシルエット
4. okinawa_silhouette.png    - 沖縄県のシルエット
5. nagano_silhouette.png     - 長野県のシルエット
6. osaka_silhouette.png      - 大阪府のシルエット
7. fukuoka_silhouette.png    - 福岡県のシルエット

推奨仕様：
========================
- 形式: PNG（透過背景推奨）
- サイズ: 300x300px 程度
- 色: 黒または濃いグレーのシルエット
- 背景: 透明または白

画像の入手方法：
========================
1. フリー素材サイトから入手
   - いらすとや (https://www.irasutoya.com/)
   - ICOOON MONO (https://icooon-mono.com/)
   - など

2. 自作する場合
   - 都道府県の地図データからシルエットを作成
   - 画像編集ソフト（GIMP、Photoshopなど）でトレース

注意：
========================
画像がない場合でも、アプリケーションは動作しますが、
画像が表示されないため、グレーの背景が表示されます。
